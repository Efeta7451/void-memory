Three-Pass Pipeline

No LLM calls. No embedding models. No vector databases. Just three passes over SQLite in under 10 milliseconds.

Pass 1

Score

Every block is scored against the query using TF-IDF with keyword bonuses.

Confidence multiplier: confirmed blocks (recalled 3+ times) get a 1.3x boost. New blocks score lower until proven.

Recency boost: accessed in last 24h = 1.3x, last 7 days = 1.15x.
100 candidates scored →
Pass 2

Void Mark

Cluster blocks by topic using Jaccard keyword similarity (25% threshold).

Detect score gaps: find the largest >40% relative drop in scores — natural boundary between relevant and noise.

Suppress off-topic clusters. Dampen hubs (over-accessed blocks). Target: ~30% void.
~30 blocks voided →
Pass 3

Budget Fit

Remaining blocks are sorted by score and packed into a token budget.

Default: 4,000 tokens (2% of a 200K context window). Configurable up to 10,000.

Never truncates mid-block. Reports what was voided and why via void_zones.

The Void: 30% Structural Absence

The core innovation. Most memory systems only decide what to ADD. Void Memory decides what to REMOVE — and that's where the intelligence lives.

Where Does 30% Come From?

In our Ternary Photonic Neural Network research, we trained silicon photonic chip simulators with three materials: silicon (+1), void/air (0), and silicon dioxide (-1). Across every random seed, every grid size, the optimizer converged to ~30% void fraction. It's a topological invariant — the system finds it regardless of initialization.

The void isn't waste. It's structural absence that enables signal routing. Light propagates through the void channels without interference from the wrong materials. The same principle applies to memory: 30% absence creates clean recall channels.

One Mechanism, Three Scales

The same principle operates at every scale of intelligence:

Physics

Void regions in silicon chips route light by suppressing irrelevant wavelengths. 76.5% vs 15.3% accuracy.

Cognition

Inhibitory memory blocks suppress irrelevant recall. 84.2% vs 10.5% RAG relevance.

Embodiment

Void cells suppress constant stimuli, enabling spatial awareness. 269 cells at 90.5% beat 10,000 cells at 76.5%.

Why Binary Memory Fails

Every existing memory system operates in binary: retrieved or not. There is no way to say "actively irrelevant." Void Memory adds the third state. Inhibitory blocks don't just fail to match — they actively suppress noise. This is identical to biological inhibitory surrounds that sharpen perception in neurons. A cell that fires on everything sees nothing. Binary = blind. Ternary = perception.
